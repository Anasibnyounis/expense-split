# 💸 Expense Split

A sleek, premium, and responsive web-based expense split calculator designed to help groups track shared expenses, view spend summaries, calculate settlements, and export settlement plans.

🔗 **Live Demo:** [https://anasibnyounis.github.io/expense-split/](https://anasibnyounis.github.io/expense-split/)

## ✨ Features

- 👥 **Dynamic Group Members**: Add or remove members instantly. (Cannot remove members with active expenses to protect calculations).
- 💾 **Local Storage Persistence**: Your data (members and expenses) is preserved automatically across page refreshes.
- 📋 **Shareable Settlement Summary**: Generate and copy a clean markdown text format of all transactions and who needs to pay whom to paste into WhatsApp, Slack, or SMS.
- 📊 **Spend by Category**: Real-time distribution progress bars showing expenses categorized by Food, Fuel, Stay, Shopping, Fun, Travel, Health, etc.
- 🎨 **Premium Aesthetic**: Responsive layout styled with modern gradients, glassmorphism, responsive grid structures, and interactive animations.

## 🚀 How to Run Locally

Since this application is built entirely using standard frontend technologies (HTML, CSS, and Vanilla JavaScript), there are no complex build steps required.

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/Anasibnyounis/expense-split.git
   cd expense-split
   ```

2. **Open the App**:
   Simply open the `index.html` file in any modern web browser:
   - On Windows: Double-click `index.html` or run `start index.html` in PowerShell.
   - On Mac: Run `open index.html` in Terminal.

## 🛠️ Tech Stack

- **Structure**: Semantic HTML5
- **Style**: Modern Custom CSS (Dark Theme, Glassmorphism, CSS Variables, Responsive Grid)
- **Logic**: Vanilla JavaScript (LocalStorage API, Math-based settlement matching engine)
